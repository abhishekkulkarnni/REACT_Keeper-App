 
 # _React.js Note Taking App_
 Google Keep Clone

## _Technologies Use :_
##### React.js, Hooks, Material UI etc.

## _How To Run :_
``` javascript
npm start
```

## _Functionality :_
##### It is clone for Google keep app.
##### You can keep your notes save here. Once complete with the tasks, delete that note from app.
